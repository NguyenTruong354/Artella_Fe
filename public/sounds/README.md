# Auction Sound System

This directory contains sound files for the auction system. You need to add the following sound files:

1. `crowd-chatter.mp3` - Ambient crowd chatter sound (loop)
2. `applause.mp3` - Crowd applause sound effect
3. `bid-success.mp3` - Sound effect for successful bids

## Sound File Requirements

### crowd-chatter.mp3
- Duration: 30-60 seconds (will be looped)
- Format: MP3
- Bitrate: 128-192 kbps
- Volume: Normalized to -14 LUFS
- Content: Subtle crowd murmur, occasional whispers

### applause.mp3
- Duration: 3-5 seconds
- Format: MP3
- Bitrate: 192 kbps
- Volume: Normalized to -14 LUFS
- Content: Medium-sized crowd applause

### bid-success.mp3
- Duration: 1-2 seconds
- Format: MP3
- Bitrate: 192 kbps
- Volume: Normalized to -14 LUFS
- Content: Short, positive chime or bell sound

## Where to Find Sound Files

You can find suitable sound effects from these sources:
1. [Freesound](https://freesound.org/)
2. [Zapsplat](https://www.zapsplat.com/)
3. [SoundBible](https://soundbible.com/)

Make sure to check the license of the sound files you use and comply with their terms of use.

## Adding Sound Files

1. Download the sound files from your chosen source
2. Convert them to MP3 format if needed
3. Normalize the volume using a tool like Audacity
4. Place the files in this directory
5. Make sure the filenames match exactly as specified above
