# Artella Frontend

Dự án React sử dụng Vite, Tailwind CSS, Framer Motion.

## Cài đặt và chạy local

```bash
npm install
npm run dev
```

## Thư mục chính
- `src/components`: Các component dùng chung
- `src/pages`: Các trang chính
- `src/styles`: Style toàn cục

## Công nghệ
- React
- Vite
- Tailwind CSS
- Framer Motion
- React Router
